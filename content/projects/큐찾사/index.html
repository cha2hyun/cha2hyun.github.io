큐찾사 (2021.09 ~)

프로젝트 소개#

🕋 (주)김치빌리아드
🧑‍💻 채수현 @cha2hyun
🏆 2021 SMART APP AWARD 생활 쇼핑분야 최우수상

100% 안심 중고큐 거래 플랫폼 큐찾사는 큐를 중고로 사고 팔때 안심하게 거래할 수 있도록 PG사 결제 시스템, NICE ID실명인증, 본인계좌확인 등 여러 안전한 시스템을 도입하였습니다.

주요기능#

  • 내큐찾기 (리스트, 상세, 서치, 알림, 결제)
  • 내큐팔기 (등록 및 결제, 알림, 정산)
  • 커뮤니티 (리스트, 상세, 등록)
  • 마이페이지

프로젝트 스택#

👉 앱 Google Play / App Store#

  • Front : React Native CLI, Typescript, Styled Component
  • Api : Django, Node.js
  • Server : AWS (EC2, Autoscaling, RDS, S3, CloudFront, ACM, Lambda)
  • Etc : Github Action, Docker

👉 웹 PC / Mobile (Beta)#

  • Front : Next.js, Typescript, Tailwind CSS
  • Api : Django
  • Etc : Github Action, Jest, ESLint, Conventional Commit Lint
