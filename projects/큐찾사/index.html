큐찾사 (2021.09 ~)

프로젝트 소개#

🕋 (주)김치빌리아드
🧑‍💻 채수현 @cha2hyun
🏆 2021 SMART APP AWARD 생활 쇼핑분야 최우수상

100% 안심 중고큐 거래 어플리케이션 큐찾사
세계적인 당구 기업 김치빌리아드가 직접 제작, 운영합니다.

<내큐찾기>#

  • 큐찾사가 100% 전 매물을 관리, 판매합니다. (사기 NO, 안심 YES)
  • 무게, 휨 상태, 도장 상태, 선골 상태와 같은 중요한 정보들을 객관적이고 일관성있게 제공합니다.
  • 새 큐 컨디션의 중고큐 프리미엄 등급의 큐를 합리적인 가격에 제공합니다.
  • 원하는 무게, 구성, 상품을 필터링하여 검색할 수 있습니다.
  • 원하는 매물이 없을 경우 즐겨찾기 해두시면 등록될 때 자동으로 푸쉬 메세지를 보내드립니다.

<내큐팔기>#

  • 큐찾사에 보내만 주시면 알아서 판매해드립니다.
  • 큐찾사가 검수 > 촬영 > 등록 > 판매 > 상담 > 정산을 진행해드립니다.
  • 큐찾사 앱과 김치빌리아드 교대점 매장에서 동시에 판매됩니다.
  • 언제든지 큐를 회수 하실 수 도 있습니다.
  • 판매 완료시에만 수수료 8%가 부과됩니다.

<커뮤니티>#

  • 더 이상 광고 게시글은 NO. 실명 인증된 회원들로 만 구성된 깨끗한 당구 커뮤니티
  • 내가 관심 있는 회원이 글을 작성하면 푸쉬 알림으로 보내드릴게요

프로젝트 스택#

👉 앱 Google Play / App Store#

  • Api : Django, Node.js
  • Server : AWS (EC2, Autoscaling, RDS, S3, CloudFront, ACM, Lambda)
  • Etc : Github Action, Docker

👉 웹 PC / Mobile (Beta)#

  • Front : Next.js, Typescript, Tailwind CSS
  • Api : Django
  • Etc : Github Action, Jest, ESLint, Conventional Commit Lint
